# 🔧 USS44 문제 해결 가이드

> 
> **작성일**: 2026-01-19

## 🔊 오디오 문제

### 1. 소리가 전혀 나지 않음

**증상**: 패드를 터치해도 아무 소리가 없음

**해결 방법**:
1. **iOS Safari**: 화면을 터치하여 AudioContext 시작
2. **무음 모드 확인**: iOS 기기의 무음 스위치 해제
3. **볼륨 확인**: 기기 볼륨 및 패드 Gain 확인
4. **Mute/Solo 확인**: 패드가 뮤트되었거나 다른 패드가 솔로 상태인지 확인

```javascript
// 디버깅: 콘솔에서 확인
const { audioContext } = useAudioStore.getState();
console.log('AudioContext state:', audioContext?.state);
// 'running'이어야 정상
```

### 2. "Audio Paused" 화면 표시 (iOS)

**증상**: 앱이 백그라운드에서 돌아온 후 오디오가 멈춤

**해결 방법**: "TAP TO RESUME" 버튼 터치

**원인**: iOS Safari는 백그라운드 복귀 시 AudioContext를 일시 중지함

### 3. 클릭/팝 노이즈

**증상**: 재생 시작/끝에 딱딱 소리

**해결 방법**:
1. 패드의 Attack/Release envelope 값 증가 (최소 1-5ms)
2. 파형 편집기에서 "Zero-Crossing Snap" 활성화
3. Auto Crop 버튼으로 무음 부분 제거

### 4. 오디오 지연/레이턴시

**증상**: 터치와 소리 사이 지연

**해결 방법**:
- 브라우저 탭을 새로고침
- 다른 탭 닫기 (CPU 부하 감소)
- Chrome/Safari 권장 (Firefox는 레이턴시 높음)

## 🎹 패드/시퀀서 문제

### 5. 패드에 샘플이 로드되지 않음

**증상**: 샘플 선택 후 패드가 비어있음

**해결 방법**:
1. 네트워크 연결 확인 (외부 샘플인 경우)
2. 브라우저 콘솔에서 에러 확인
3. IndexedDB 초기화: Settings → Init All

### 6. 시퀀서가 재생되지 않음

**증상**: Play 버튼 눌러도 시퀀서 동작 안함

**해결 방법**:
1. 스텝이 활성화되었는지 확인 (빨간 LED 켜짐)
2. 해당 패드에 샘플이 로드되었는지 확인
3. BPM이 너무 낮지 않은지 확인 (최소 20)

### 7. 시퀀서 타이밍이 불안정

**증상**: 박자가 흔들림

**해결 방법**:
- CPU 집약적 작업 중지
- 브라우저 탭 최소화하지 않기
- 백그라운드 앱 종료

## 📱 UI/레이아웃 문제

### 8. 화면이 잘림/깨짐

**증상**: 일부 UI가 화면 밖으로 벗어남

**해결 방법**:
1. 기기 회전 후 다시 회전
2. 브라우저 새로고침
3. 주소창 숨기기 (전체 화면)

### 9. 터치가 안 먹힘

**증상**: 패드 터치 반응 없음

**해결 방법**:
1. 다른 모달이 열려있지 않은지 확인
2. Clone Mode가 활성화되었는지 확인 (Cancel 버튼)
3. 페이지 새로고침

### 10. 노브가 움직이지 않음

**증상**: 노브 드래그 반응 없음

**해결 방법**: 위/아래로 드래그 (좌우 아님)

## 💾 저장/로드 문제

### 11. 프로젝트가 저장되지 않음

**증상**: 저장했는데 새로고침 후 사라짐

**해결 방법**:
1. 프라이빗/시크릿 모드 사용 중인지 확인 (IndexedDB 제한)
2. 저장소 용량 확인 (대용량 샘플 주의)
3. JSON 내보내기로 백업

### 12. 가져오기 실패

**증상**: JSON 파일 import 오류

**해결 방법**:
1. 올바른 USS44 JSON 포맷인지 확인
2. 파일 인코딩 UTF-8 확인
3. 파일이 손상되지 않았는지 확인

## 🎤 녹음 문제

### 13. 마이크 권한 거부됨

**증상**: UltraSample 모드 진입 실패

**해결 방법**:
1. 브라우저 마이크 권한 허용
2. 시스템 설정에서 마이크 권한 확인
3. HTTPS 연결 확인 (HTTP에서는 마이크 차단)

### 14. 녹음 품질이 낮음

**증상**: 녹음된 소리가 왜곡되거나 노이즈

**해결 방법**:
1. 마이크 입력 레벨 조정
2. 조용한 환경에서 녹음
3. 외부 마이크 사용 고려

## 🔄 리셋/초기화

### 완전 초기화 방법

1. **Settings → Init All**: 공장 초기화
2. **브라우저 데이터 삭제**:
   - Chrome: 설정 → 개인정보 → 사이트 데이터 삭제
   - Safari: 설정 → Safari → 기록 및 웹사이트 데이터 지우기

### IndexedDB 직접 삭제 (개발자용)

```javascript
// 브라우저 콘솔에서
indexedDB.deleteDatabase('uss44-db');
location.reload();
```
